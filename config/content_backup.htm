如何開啟9443

開啟小黑窗後依序輸入

cd tmp

cd cd2021

cd cmsimde

python wsgi.py

接著到網頁上芳欄位輸入

https://localhost:9443

如何開啟8444

打開 data → tmp → cd2021

將 http-server.py 拖至小白窗

按 Tools  → Go

如何上傳

先進到cd tmp → cd cd2021

git add .

git commit -m "此更新的主旨"

git push

輸入帳號(學號)

輸入密碼
